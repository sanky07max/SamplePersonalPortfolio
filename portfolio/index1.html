Project 2

Titanic survival Project

This is a beginner’s project on Kaggle that is best for you if you are just getting started with ML projects. This deals with the Titanic disaster that is one of the most famous in marine history. All you have to do in the project is predict which passengers survived the Titanic shipwreck based on data such as their age, gender, socio-economic class, etc. For example, it is highly probable that someone rich from the first-class survived as compared to someone from the third class. You can use the Titanic dataset on Kaggle for this project. And just to make things more interesting, this dataset contains real data based on the people who died and survived the actual Titanic disaster.
